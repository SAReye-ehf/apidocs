## Update an existing area

### HTTP request

`PATCH/PUT https://api.sardynamics.com/areas/<ID>`

### Parameters

| Key         | Type    | Description      |
| ----------- | ------- | ---------------- |
| name        | String  | Name of the area |
| active      | Boolean |                  |
| description | String  |                  |
| lat         | String  |                  |
| lng         | String  |                  |
| zoom        | Integer |                  |
